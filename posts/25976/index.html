Hexo博客的搭建

搭建博客的由来？

加入全球最大的同性交友网站 - github 这个平台距离今天也有一年多(虽然活跃度极低)。又想起刚入职时，谈及自己的未来规划，信誓旦旦的说想要搭建自己的博客，迟了一点但绝对不晚。因为我相信：一切伟大的行动和思想，都有一个微不足道的开始。

搭建博客的过程？

参考文章：
Hexo搭建博客
唠师傅仔细教你建站系列2—本地搭建个人博客

搭建博客的目的？

  • 记录学习过程、心得体会，工作中遇到的问题及解决方案
  • 搭建知识网络体系，学习强化自身的 Java 技术栈
  • 快速回忆技术和知识点
  • 想进大厂会提供一些便利

第一个总结

  • 刚接触写博客这件事，主题选择的很简单，后面有需要会把页面做的更漂亮
  • MarkDown2 刚开始使用，只会几个简单的样式，后面会继续学习补充
